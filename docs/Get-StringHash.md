# Get-StringHash

## SYNOPSIS
Returns a hash of a string value

## SYNTAX

```
Get-StringHash [-String] <String> [-HashName] <String> [<CommonParameters>]
```

## DESCRIPTION
Returns a hash of a string value

## EXAMPLES

### EXAMPLE 1
```
Get-StringHash -string "qwerty" -hashname SHA1
```

## PARAMETERS
